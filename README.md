
# Automação utilizando o curso Testando com Inteligência (Artificial)


Treinamento de IA utilizando ChatGPT gerando scripts automatizados em Cypress


## Referência

 - [Cypress](https://www.cypress.io/#create)
 - [ChatGPT](https://chatgpt.com/)
 - [WCAquino.me](https://wcaquino.me/)


## Instalação

Instale Node v18.19.0
```bash
  nvm install 18.19.0
  nvm use 18.19.0

  node -v
```

Instale o npm v10.9.3
```bash
npm install npm@10.9.3 -g

npm -v
```

Instale as dependências que estão no pacote
```bash
  npm install -D
```
